# Бездраное Очко (alpha)

## Описание проекта

Бездраное Очко - карточная игра в жанре 21 очко. Игрок играет против дилера и старается набрать сумму карт как можно ближе к 21, не превышая это значение.

Проект выполнен в рамках индивидуального учебного проекта.

---

## Основные механики

- Реализация карточной игры 21 очко
- Создание и перемешивание стандартной колоды из 52 карт
- Подсчет очков по правилам игры
- Простая логика дилера
- Экран результата раунда (победа, поражение, ничья)

---

## Используемые технологии

- Godot Engine 4.6
- GDScript
- Разрешение 1920x1080
- Структура проекта построена на сценах Godot и главном игровом контроллере
- JSON для хранения файлов игры

---

## Структура игры

В игре есть несколько экранов:

1. Главное меню
   - запуск новой игры
   - меню выигранных предметов

2. Игровой экран
   - раздача карт
   - действия игрока (взять карту или остановиться)

3. Экран результата
   - показ результата раунда (победа, поражение, ничья)
   - переход дальше
   - отображение полученных наград в случае победы

---

1. Открыть проект в Godot 4.6

В репозитории есть папка:


/build


В ней находятся готовая сборка:
- .exe файл
- .console.exe файл с консолью для дебага

---

## Как устроена игра

Основная логика находится в главном контроллере игры. Он отвечает за:

- ходы игрока и дилера
- работу с колодой
- состояние игры

### Формат карт

Колода представлена массивом:
["ha", "d6", "s10", ...]
Карты представлены строками, например "c10" - десятка крестей


где:
- первая буква - масть
- число - значение карты

Главный контроллер игры парсит номинал карты при операциях с ними
